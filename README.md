# IMP_GCN

Fan Liu, Zhiyong Cheng*, Lei Zhu, Zan Gao and Liqiang Nie*. Interest-aware Message-Passing GCN for Recommendation. The Web Conference 2021 (WWW'21), Ljubljana, Slovenia, 2021 (“*”= Corresponding author)
